About

此內容管理系統以 https://github.com/mdecourse/cmsimde 作為 submodule 運作, 可以選定對應的版本運作, cmsimde 可以持續改版, 不會影響之前設為 submodule, 使用舊版 cmsimde 模組的內容管理相關運作.TEST

利用 cmsimde 建立靜態網誌方法:

1. 在 github 建立倉儲, git clone 到近端

2. 參考 https://github.com/mdecourse/newcms, 加入除了 cmsimde 目錄外的所有內容

以 git submodule add https://github.com/mdecourse/cmsimde cmsimde

建立 cmsimde 目錄, 並從 github 取下子模組內容.

3.在近端維護時, 更換目錄到倉儲中的 cmsimde, 以 python wsgi.py 啟動近端網際伺服器.

動態內容編輯完成後, 以 generate_pages 轉為靜態內容, 以 git add commit 及 push 將內容推到遠端.

4. 之後若要以 git clone 取下包含 submodule 的所有內容, 執行:

git clone --recurse-submodules https://github.com/mdecourse/newcms.git

Develop

https://github.com/mdecourse/cmsimde 的開發, 可以在一個目錄中放入 cmsimde, 然後將 up_dir 中的內容放到與 cmsimde 目錄同位階的地方, 使用 command 進入 cmsimde 目錄, 執行 python wsgi.py, 就可以啟動, 以瀏覽器 https://localhost:9443 就可以連接, 以 admin 作為管理者密碼, 就可以登入維護內容.

cmsimde 的開發採用 Leo Editor, 開啟 cmsimde 目錄中的 cmsimde.leo 就可以進行程式修改, 結束後, 若要保留網際內容, 只要將 cmsimde 外部的內容倒回 up_dir 目錄中即可後續對 cmsimde 遠端倉儲進行改版.

init.py 位於  up_dir 目錄, 可以設定 site_title 與 uwsgi 等變數.

執行器(Actuators)


https://cyberbotics.com/doc/guide/actuators

Webots can simulate a lot of standard robotic actuators.
Webots可以模擬許多標準的機器人執行器
Brake-Simulates a mechanical braking system.
致動器-模擬機械致動功能
Connector-Simulates a mechanical and breakable docking system.
連接器-模擬機械和易碎的對接系統
Display-Simulates a computer screen.
顯示 -模擬計算機屏幕
Emitter-Simulates radio, serial or infra-red emitters sending data to other robots.
發訊器-模擬無線電，串行或紅外發射器，將數據發送到其他機器人。
LinearMotor-Simulates a linear motor.
線性馬達-模擬線性電動機
LED-Simulates a light emitting diode (LED).
LED-模擬發光2極體(LED)
Muscle-Simulates the shape of an artificial muscle.
肌肉-模擬人造肌肉的形狀
Pen-Simulates a pen able to draw on any graphical geometric primitive.
鋼筆-模擬能夠在任何圖形幾何圖元上繪製的筆
Propeller-Simulates a marine or aircraft propeller
螺旋槳-模擬船用或飛機螺旋槳
RotationalMotor-Simulates a rotational motor.
旋轉馬達-模擬旋轉電機
Speaker-Simulates a loudspeaker device.
揚聲器-模擬揚聲器設備
Track- Simulates a track for conveyor belts or tank robots.
跟蹤-模擬傳送帶或儲罐機器人的軌道。

感測器(sensors)


https://cyberbotics.com/doc/guide/sensors

Webots can simulate a lot of standard robotic sensors.
Webots可以模擬許多標準的機器人傳感器
Accelerometer-Simulates an accelerometer sensor which measures the relative accelerations.
加速計-模擬測量相對加速度的加速度傳感器。
Camera-Simulates an RGB camera, a linear camera, a gray-scale camera, a fish-eye camera or a smart camera with multiple special effects including noise, depth of field, motion blur or lense flares.
相機-模擬具有多種特殊效果（包括噪聲，景深，運動模糊或鏡頭光暈）的RGB相機，線性相機，灰度相機，魚眼鏡頭或智能相機
Compass-Simulates a magnetic sensor which measures the relative direction to the north.
羅盤-模擬一個磁傳感器，該傳感器測量向北的相對方向。
DistanceSensor-Simulates a distance measuring sensor based on infra-red light, sonar echo, or laser beam.
距離傳感器-基於紅外光，聲納迴聲或激光束模擬測距傳感器。
GPS-Simulates a positioning sensor which measures the absolute position in Webots coordinate system or in the WGS84 coordinate system
全球定位系統-模擬一個定位傳感器，該傳感器測量Webots坐標系或WGS84坐標系中的絕對位置。
Gyro-Simulates a gyroscope sensor which measures the relative angular velocities.
陀螺儀-模擬測量相對角速度的陀螺儀傳感器。
InertialUnit-Simulates a sensor which measures the relative roll, pitch and yaw angles.
慣性單位-模擬一個傳感器，該傳感器測量相對的側傾角，俯仰角和偏航角。
Lidar-Simulates a laser-scanner sensor also known as lidar
積光雷達-模擬也稱為激光雷達的激光掃描儀傳感器。
PositionSensor-Simulates a sensor which can monitor a joint position, such as an encoder or a potentiometer
位置傳感器-模擬可以監視關節位置的傳感器，例如編碼器或電位計。
Radar-Simulates a radar sensor.
雷達-模擬雷達傳感器。
RangeFinder-Simulates a depth camera also known as range-finder.
測距儀-模擬深度相機，也稱為測距儀。
Receiver-Simulates radio, serial or infra-red receiver receiving data from other robots.
接收器-模擬從其他機器人接收數據的無線電，串行或紅外接收器。
TouchSensor-Simulates a bumper or a force sensor.
觸控感應器-模擬保險槓或力傳感器。
